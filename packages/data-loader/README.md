data-loader
===========

_in development_

A Custom Element fetching data declaratively.

Will load the data at the url defined in the `src` attribute of a
`source` element that it contains.

## Usage

```html
<data-loader>
  <source src="https://www.example.com/some/data">
</data-loader>
```
